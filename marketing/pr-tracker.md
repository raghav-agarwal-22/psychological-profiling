## Send Instructions

**Before sending — complete these steps:**
1. Replace `[Founder name]` in all pitches with actual name
2. Add real user/revenue metrics to TechCrunch pitch (#8) — format: "X users, $Y MRR, Z% MoM growth"
3. Check innermindhealing.com is live and assessment works end-to-end

**Priority send order (highest ROI first):**
1. **Ben's Bites** — Ben reads personally, responds fast, AI audience, short pitch
2. **TLDR Newsletter** — millions of subscribers, short pitch, high upside
3. **The Neuron AI** — Claude use case angle is perfect fit
4. **AI Breakfast** — short pitch, AI audience, easy yes/no
5. **Greater Good** — article pitch, perfect audience fit for conversion
6. **The Hustle** — data-first pitch, needs metrics
7. **Big Think** — opinion piece, needs time to write if accepted
8. **Psyche Magazine** — long-form essay, lowest effort to pitch but high effort to write
9. **TechCrunch** — cold outreach is hard, needs strong metrics
10. **Psychology Today** — contributor application, multi-week process, start in parallel

**Timing:**
- Send pitches 1-4 (AI newsletters) on same day — Tuesday/Wednesday morning (best open rates)
- Send pitches 5-8 (editorial/features) 2-3 days later
- Start Psychology Today contributor application independently (takes 2-4 weeks)
